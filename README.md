# maglight #

```
Usage: maglight [device] [command] <args>

Devices:
- [k]eyboard
- [d]isplay

Commands:
- [g]et (default)    Get a device's brightness level

- [s]et <num>        Set a device's brightness to number
- [a]dd <num>        Increase a device's brightness by number
- s[u]b <num>        Decrease a device's brightness by number

% [p]ercent <per>    Set a device's brightness to percentage
% [i]nc <per>        Increase a device's brightness level by percentage
% [d]ec <per>        Decrease a device's brightness level by percentage
```
